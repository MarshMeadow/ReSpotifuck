# ReSpotifuck - Spotify WebView App For Android (Ad-Free) Open Source.
About Spotifuck: is essentially a webview wrapper of spotify.com (Ad-Free)

This is a minimal Android app that loads a Spotify web URL inside a WebView with no ads.

OG Repo: https://github.com/MarshMeadow/ReSpotifuck-Src-Clone
